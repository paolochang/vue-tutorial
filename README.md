# Vue Tutorial

## Check installation

```sh
$ vue --version
```

## Installation

```sh
# make sure remove all vue cli
$ sudo npm uninstall --global vue-cli
$ sudo npm uninstall --global @vue/cli
# install the latest version of vue cli
$ sudo npm install --global vue
$ sudo npm install --global @vue/cli@latest
```

## Update Vue version

```sh
$ npm update -g @vue/cli
```
